# How to use.
### prerequisite 
- [GNU-stow](https://www.gnu.org/software/stow/)  utility. 
- neovim nightly version ( I am managing neovim versions using [bob](https://github.com/MordechaiHadad/bob) )
- And other extras...

### commands 
- Clone the repo in the home directory.
```git clone https://github.com/ManogyaDahal/config```
- use Stow command inside the cloned directory  
```stow .```
- Then you are good to go with the config...
